# WordPress-Blog
WordPressを使ったブログサイト制作

## My Portfolio
 - ２カ月間職業訓練校でWordPressについて学びました。そこで得た知識をもとに**My Portfolio**という名前でブログサイトを作成しました。
 - PHPでの業務経験を生かして、TOPページは独自にカスタマイズしました。
 - 主に、勉強した内容を自主的に記録していくために作成しましたので一般公開はする予定はありません。
